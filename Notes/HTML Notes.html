HEAD

You can construct your website style and global web setting in the head

  • [title]~> defines the title in the browser toolbar
  • [base]~> specifies the URL/target or storage you want to refer to if there is image
  • [link]~> used to link file to this page
    ~> rel specify the relationship between the current document and the linked document, e.g. rel="stylesheet"
    ~> type specify the type of the linked document, e.g. type="text/css"
  • [style]~> defines the style information for the HTML
  • [meta]~> provides data about the HTML, can also used to refresh
  • e.g. meta http-equiv="refresh" content="30"
  • [scritp]~> define a client-side script

PARAGRAPHS AND FORMATS

  • [p]~> used to indicate that it is another paragraph.
  • [hr/]~> for horizontal line
  • [br/]~> for line break
  • [b]~> for bold text
  • [i]~> for italic text
  • [sub] for subscripted text
  • [sup] for superscripted text
  • [pre] preserves the original style in spacing, good for displaying computer code
  • [code] displays as computer code, often used to programming code.
  • [abbr] for abbreviation with [title=""]
  • [q] means quotation
  • [del] for strikethrough the text
  • [ins] for underline the text
  • [mark] for highlight the text
  • [!--]~> for comments

  • To display < or >
  • you have to type lt and gt with & in front of them

  • class
  • Syntax: [element class="classname"]
  • this attribute specifies one or more classmates for an element
  • to specify more than one classname, you have to separate them with a space

LINKS

Syntax: [a href="URL"]Text[/a]

  • with id="" for anchor in the website which can be referred later
  • with title="" for pop-up when you move the mouse on it
  • if you want to go to previous anchor, then use href="#id"
  • you can also refer to the page in the same folder
    • This will lead you to the second page.
  • if you link to email, then use mailto:xxx@gmail.com in href
  • you can use 2 attributes, at least one space bewtween them

IMAGES

Syntax: [img src="url" alt="some_text"]

  • [img]~> for image
  • with src="url" the source of your image
  • with alt="some_text" to display text if the image cannot be displayed
  • with height="number"
  • with style="float:left" to float the image to left of the text
  • with usemap="#map_name" to use the image as map later on

  • [map]~> defines an image-map
  • Syntax: [map name=""] which is previous defined in the img
  • [area]~> defines a clickable area inside an image-map

  • with border="", the priority will be CSS, attribute inside table

FORMS AND INPUTS

Syntax: [form] [input type=""] OR [form] [select name=""]

  • For input type=""
  • "text" defines a one-line input field that a user can enter text into
  • "password"
  • "radio" let user select only one option
  • "checkbox" let user select zero or more options
  • "button"
  • follow up with name="" and value="" for variable name and identify the input

  • For select name=""
  • the name is the variable name
  • follow up with [option value="item1"]item1[/option]
  • you can add [optgroup label=""] to group related items

  • [textarea rows="number" cols="number"]
  • allow you to write something

  • You can use [fieldset] to bracket as personal information
  • And [legend] for the title

  • Attributes for forms
  • name=""
  • action="URL" specifies where to send the form-data wen a form is submitted
  • method="get/post" specifies the HTTP method to use when sending form-data
