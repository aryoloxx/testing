CSS SYNTAX

Syntax: Selector {Property1:Value1; Property2:Value2;}

  • the front part is called selector
    • selector points to the HTML element you want to style
    • you can also use id attribute with #; e.g. #MyFamily
    • you can also use class attribute with .; e.g. p.important
  • the later part is called declaration

  • /*Here is the comments*/ only applicable inside the declaration

  • you can group the selector with the same style by comma

  • Insert External CSS Style Sheet
  • Syntax: [link rel="name_you_decide" type="text/css" href="URL"]
  • place between [head], but not between [style]
  • the file should not contain any html tags and saved with a .css extension

  • if there is more than one style for the same element, then for the priority will be the followings:
  • inline, internal style sheet, external style sheet
  • if the external style sheet is placed after the internal style sheet, then it will override the later

BACKGROUND

  • color, image, repeat, attachment, position

  • background-color: #485C99;
  • background-image: url("name.gif");
  • background-repeat: norepeat/repeat-x;
  • background-position: right top;
  • for a shorthand, you can use background with the above order

TEXT

FONT

LINK

  • Selector should be in the form of the followings:
  • a:link/visited/hover {}
  • e.g. a:link {color:#00FF00;}
  • e.g. a:visited {color:#00FF00;}
  • e.g. a:hover {color:#FFCC00;}
