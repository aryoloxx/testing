SYNTAX

  • For data type
  • Syntax: var name = number/string/array/object
  • var is used to create new variable, you can declare many variables in one statement by commas
  • variable names must begin with a letter and case sensitive

  • Syntax: function function_name(parameter1, parameter2){program progress}

LOOP

  • For Loop
    Syntax: for (executed before loop; condition for the loop; executed each time) {
         code block;
    }
    rmb use semi-colon; (but not comma,) to separate the 3 items
    you can leave the first blank or initiate more than one var in it
    the 3rd statement can be blank and you increment i inside the loop

    • e.g. List out the cars:

      var cars = ["BMW","Volvo","Audi","Ford"];
      var text = "";
      for (var i = 0; i < cars.length; i++) {
           text += cars[i] + "line break tag";
      }

    • e.g. For loop on object
      you can directly use the index of the object

      var text = "";
      var person = {fname:"John", lname:"Doe", age:25};
      for (var x in person) {
           text += person[x] + " ";
      }

  • While Loop
    Syntax: while (condition) {
         code block;
    }
    rmb to increment, otherwise infinity loop
    • e.g. Listing:

      var text="";
      var i = 0;
      while (i<10) {
           text += "The number is " + i + "line break tag";
           i++;
      }

  • Do/While Loop
    Syntax: do {
         code block;
    }
    while (condition);

  • Break and Continue statement
    Syntax: break labelname; continue labelname;
    break will jump out the loop immediately, (without labelname can only be used inside loop or switch)
    continue will continues with the next iteration in the loop, (can only be used inside loop0

MORE ON OBJECT

  • Everything in JavaScript can be object, e.g. primitive data, function
  • Properties are values associated with an object, objectName.property
  • Methods are actions which can be performed on object, objectName.methodName

  • you can create object by the following syntax
    • Using Object Literal
      var objectName = {
          prop1:value1,
          prop2:value2
      };
    • Using Keyword New
      var objectName = new Object();
      objectName.prop1 = value1;
      objectName.prop2 = value2;
    • Using Object Constructor
      function objectName(prop1, prop2){
          this.prop1 = prop1;
          this.prop2 = prop2;
      }
      var objectName1 = new objectName(value1, value2);
      var objectName2 = new objectName(value1, value2);

  • this is the object that "owns" the current code
    e.g. when used in a function, the value of this is the object that owns the function
    when a function is called without an owner object, the value of this becomes the global object which is the browser window
  • for the function belonging to an object, this refer to the object

  • !!Mutable objects are addressed by reference, not by value
    e.g. if y is an object and var x = y; we say x is a pointer to y, any changes to y will also change x

      • var y = new person{firstName:Eric, lastName:Liu, Age:23, Sex:Male};
      var x = y;
      x.age = 30;       here y.age will also be changed to 30

  • for ... in loop
    Syntax: for (variable in loop) {
        code block
    }
    • e.g. List out the element in object

      person = {fname:"John", lname:"Doe", age:25}

  • Add and Remove element from the object
    Syntax: objectName.newProperty = value;
    Syntax: delete objectName.property;
    while you delete the property, the value will be deleted at the same time

  • To use the object method
    Syntax: objectName.method();

HTML DOM (Document Object Model)

With the HTML DOM, JavaScript can access and change all the elements of an HTML document.

  • Syntax:
    document.getElementById(id).innerHTML = newHtml;
    document.getElementById(id).attribute = newValue;
    document.getElementById(id).style.property = newStyle;
  • you can also use
    getElementsByTagName, getElementByClassName
  • everything in an HTML document is a node
    the top node is called root, everything node must have one parent, except the root
    you can get other content with the usage of childNodes[index].nodeValue
    var x = document.getElementById("demo").childNodes[0].nodeValue;
  • Reminder: [title]DOM Tutorial[/title], the element title does not contain text, but contain a text node with value "DOM Tutorial"

    • To create a new node of paragraph
      Syntax
      document.createElement("nameOfNewElementOrTagName")
      document.createTextNode("text") will help you create text
      if you use the previous defined tag name like h1, the style will be inherited

    • To insert, remove, replace node
      Syntax:
      x.appendChild(y) will append y as child of x at the last position
      x.insertBefore(y,z) will insert y into x before z
      Syntax:
      x.removeChild(y) will remove y from x
      When you don't know the parentNodeName, you can use tick parentNode
      e.g. var child = document.getElementById("demo");
      child.parentNode.removeChild(child);
      Syntax:
      para.replace(newChild,preChild) will replace preChild by newChild in para

    • Example to insert node

      var para = document.createElement("p");
      var node = document.createTextNode("This is the new paragraph");
      para.appendChild(node);
      document.getElementById("div1").appendChild(para);

  • To create a nodeList
    e.g. var pElements = document.getElementsByTagName("p");
  • A much better way to find the element in the page
    Syntax:
    document.querySelector("p") will give you the first one; you can also use "body p"; # is fo id
